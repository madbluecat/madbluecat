mysql cdc table

基于mysql的存储插件，与mysql的binlog开发了一个mysql cdc table。可以在mysql select binlog中表的变化数据

__op

0 – delete data

1 – insert data

2 – update before data

3 – update after date

__gtid

全局事务id

__tm

row event 发生的时间

支持了所有类型
